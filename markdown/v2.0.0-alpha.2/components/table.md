# Table

A table allows to arrange data into rows and columns of cells.

<hr />

## Default table

```html
<table class="table">
  <thead>
    <tr>
      <th>#</th>
      <th>First</th>
      <th>Last</th>
      <th>Framework</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Jordan</td>
      <td>Walke</td>
      <td>React</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Evan</td>
      <td>You</td>
      <td>Vue.js</td>
    </tr>
    <tr>
      <td>3</td>
      <td>Misko</td>
      <td>Hevery</td>
      <td>Angular</td>
    </tr>
  </tbody>
</table>
```

## Alignment

### Top

Use class `.table-align-top`.

```html
<table class="table table-bordered table-align-top">
  <thead>
    <tr>
      <th>#</th>
      <th>Name</th>
      <th>Description</th>
      <th>Framework</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Jordan Walke</td>
      <td>React (also known as React.js or ReactJS) is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of individual developers and companies.</td>
      <td>React</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Evan You</td>
      <td>Vue.js (commonly referred to as Vue; pronounced /vjuː/, like view) is an open-source JavaScript framework for building user interfaces and single-page applications.</td>
      <td>Vue.js</td>
    </tr>
    <tr>
      <td>3</td>
      <td>Misko Hevery</td>
      <td>Angular (commonly referred to as "Angular 2+" or "Angular v2 and above") is a TypeScript-based open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations. Angular is a complete rewrite from the same team that built AngularJS.</td>
      <td>Angular</td>
    </tr>
  </tbody>
</table>
```

## Combine text utilities

```html
<table class="table table-bordered">
  <thead>
    <tr>
      <th style="width: 140px;">Truncate</th>
      <th>Default</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td class="text-truncate">React (also known as React.js or ReactJS)</td>
      <td>React is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of individual developers and companies.</td>
    </tr>
    <tr>
      <td class="text-truncate">Vue.js (commonly referred to as Vue; pronounced /vjuː/, like view)</td>
      <td>Vue.js is an open-source JavaScript framework for building user interfaces and single-page applications.</td>
    </tr>
    <tr>
      <td class="text-truncate">Angular (commonly referred to as "Angular 2+" or "Angular v2 and above")</td>
      <td>Angular is a TypeScript-based open-source web application framework led by the Angular Team at Google and by a community of individuals and corporations. Angular is a complete rewrite from the same team that built AngularJS.</td>
    </tr>
  </tbody>
</table>
```
